# parse-marketplace-privalia
парсинг маркетплейса privalia.es - маркетплейс закрытых продаж, доступен после авторизации.
Задача была - использовать токен авторизации из кук, и по нему настроить парсинг контента, с сохранением в локальный mysql.
Для того чтобы иметь возможность показывать предлагаемый ассортимент на сайте-зеркале.
Сохраняются все бренды, id, описания, и изображения.
